# SmartDoc

## Overview
SmartDoc is a sophisticated Java project designed to help users manage and organize their documents efficiently. With a user-friendly interface and robust functionality, SmartDoc aims to streamline document handling and enhance productivity.

## Features
- **Document Organization**: Easily sort and categorize documents.
- **Search Functionality**: Quickly find documents using keywords.
- **Collaboration Tools**: Share documents with team members and collaborate in real-time.
- **File Encryption**: Keep your documents secure with encryption.
- **User Permissions**: Control access to documents based on user roles.

## Installation
To install SmartDoc, follow these steps:
1. Clone the repository:
   ```bash
   git clone https://github.com/nitishshukla121/SmartDoc.git
   ```
2. Navigate to the project directory:
   ```bash
   cd SmartDoc
   ```
3. Build the project using Maven:
   ```bash
   mvn clean install
   ```
4. Run the application:
   ```bash
   java -jar target/smartdoc.jar
   ```

## Usage
After installation, you can start using SmartDoc:
- **Create a new document**: Click on 'New Document' from the dashboard.
- **Upload files**: Drag and drop files into the application.
- **Edit documents**: Open any document and make changes as needed.
